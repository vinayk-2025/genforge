### Prompt:  
> "Generate a Python script that reads a CSV file and prints the average of a numeric column."

**Instructions**:  
1. Run the prompt on at least **two platforms**  
2. Compare:  
   - Syntax correctness  
   - Output formatting  
   - Error handling  
   - Comments and readability  
3. Paste results into a markdown table  
4. Reflect on which tool offered the most usable code and why
