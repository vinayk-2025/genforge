
# Storage Requirements for GenAI Agentic Course


## Tool-wise Disk Space Estimates

| Tool / Component            | Approx. Disk Space Needed         |
|----------------------------|-----------------------------------|
| Python + libraries         | 1-2 GB                             |
| Node.js + npm packages     | 500 MB - 1 GB                      |
| XAMPP (Apache, MySQL, PHP) | 1-2 GB                             |
| Editors (VS Code, PyCharm, Notepad++ etc.) | 500 MB - 2 GB      |
| Gemini CLI                 | ~100 MB                            |
| LMStudio (base install)    | 1-2 GB                             |
| Ollama (base install)      | 2-3 GB + 10-20 GB per model        |
| Git + Git Bash             | 500 MB                             |
| Miscellaneous tools/configs| 1-2 GB                             |


## Recommended Free Space for 10 Projects

- Baseline (10 GB x 10 projects): 100 GB  
- With larger models (13B+): 150-200 GB  
- Comfortable buffer: 200 GB or more


## Summary

- Reserve 100 GB minimum for 10 projects.  
- Plan for 150-200 GB free space if using larger models.  
- Manage models and datasets smartly to avoid running out of space.  
- With proper planning, students can complete all 10 agentic apps across 33 lectures without storage issues.
