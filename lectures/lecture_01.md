## Glossary of Key Terms
| Term              | Definition |
|-------------------|------------|
| GenAI             | Generative AI - models that create text, images, code, or other content |
| Agentic AI        | AI systems that act autonomously using tools, memory, and reasoning |
| Prompt            | Instruction or input given to an AI model |
| Zero-shot         | Prompt with no examples |
| Few-shot          | Prompt with a few guiding examples |
| Chain-of-Thought  | Step-by-step reasoning prompt style |
| LangChain         | Framework for agentic LLM workflows |
| Ollama            | Local LLM runtime |
| Gemini CLI        | Command-line interface for Gemini models |
| Streamlit         | Python framework for GenAI dashboards |
| XAMPP             | MySQL-backed local server stack |
| requirements.txt  | File listing Python packages for reproducible environment setup |


## Student Exercises
- Complete environment setup and verify installations  
- Clone the starter repo and organize your local workspace  
- Explore prompt delivery via web, CLI, Python script, and API  
- Deploy a sample GenAI app to Streamlit Cloud  
- Create a GitHub Pages site showcasing one prompt  
- Submit screenshots of setup and deployment via LMS or GitHub issue  
- Validate Python, Node.js, and local LLM runtimes  
- Practice reproducibility with `requirements.txt` and GitHub integration  
- Test LangChain, Gemini CLI, Ollama, and Streamlit workflows  


## Instructor Notes
- Emphasize reproducibility and open-source fluency from Day 01  
- Encourage students to ask setup questions early to avoid blockers  
- Use this session to build excitement and establish a collaborative tone  
- Highlight the importance of prompt hygiene, folder structure, and naming conventions  
- Reinforce the value of public deployment and documentation as part of professional readiness  
- Showcase prompt variations and their impact
