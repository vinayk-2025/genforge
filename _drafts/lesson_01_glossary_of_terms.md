# Lesson 01 - Glossary of Terms  
**Orientation & Environment Setup**

This glossary defines key terms introduced in Lecture 01. These terms support your understanding of GenAI workflows, tooling, and professional habits. Use this as a reference throughout the course.

---

| Term                         | Description                                                                 |
|------------------------------|-----------------------------------------------------------------------------|
| Generative AI                | AI systems that create new content such as text, images, or code            |
| Prompt Engineering           | Crafting inputs to guide GenAI models toward desired outputs                |
| Agentic Systems              | AI systems that act autonomously with tools, memory, and goal-driven logic  |
| Modular Curriculum           | A course structure built from reusable, standalone learning units           |
| Value-Added Learning         | Skill-building beyond core syllabus, focused on industry readiness          |
| Local-First Development      | Running GenAI tools on your own machine before cloud deployment             |
| Open-Source Tooling          | Software that is freely available for use, modification, and sharing        |
| Reproducibility              | Ability to recreate results using consistent inputs and environments        |
| Version Control              | Tracking changes to code or content using tools like Git                    |
| Python                       | A programming language widely used in AI and data science                   |
| Streamlit                    | A Python framework for building interactive web apps                        |
| LangChain                    | A framework for building GenAI workflows using language models              |
| Hugging Face                 | A platform for sharing and using open-source AI models                      |
| Ollama                       | A tool for running LLMs locally on your machine                            |
| Gemini CLI                   | A command-line interface for interacting with Google's Gemini models        |
| Prompt Delivery Workflow     | Method used to send prompts to GenAI models (Web UI, CLI, Python, API)      |
| Web UI                       | A browser-based interface for interacting with GenAI models                 |
| CLI                          | Command Line Interface - used to run shell commands and scripts             |
| Python API                   | A way to send prompts using Python code and HTTP requests                   |
| GitHub                       | A platform for hosting code and collaborating with others                   |
| GitHub Pages                 | A service for hosting static websites from GitHub repositories              |
| Streamlit Cloud              | A platform to deploy Streamlit apps online                                 |
| Setup Checklist              | A step-by-step guide to install and verify required tools                   |
| Starter Repo                 | A pre-configured GitHub repository for beginning your GenAI projects        |
| Deployment                   | Publishing your GenAI app or documentation to a public platform             |
| Documentation Hygiene        | Writing clear, consistent, and professional project notes                   |
| CC BY-NC 4.0                 | A Creative Commons licence allowing sharing with attribution, non-commercial use only |
