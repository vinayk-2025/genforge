# Glossary â€“ Agentic AI vs Autonomous Systems
**Reference for Lecture 3A and 4B**

This glossary clarifies the distinction between agentic AI and autonomous systems. It supports student understanding of how goal-driven agents evolve into systems capable of sustained, independent operation.

---

## Comparative Table

| Concept            | Agentic AI                                           | Autonomous Systems                                      |
|--------------------|------------------------------------------------------|----------------------------------------------------------|
| Definition         | AI that takes actions using tools, memory, and reasoning | Systems that operate independently over time with minimal human input |
| Scope              | Task-oriented, often short-lived or bounded          | Long-term, environment-aware, self-sustaining            |
| Human Involvement  | May require human prompts, goals, or oversight       | Designed to function with minimal or no human intervention |
| Examples           | Tool-using LLM agents, RAG pipelines, copilots       | Self-driving cars, autonomous drones, robotic warehouses |
| Dependencies       | Often rely on APIs, toolchains, or external memory   | Integrate sensing, planning, and control internally      |
| Evaluation         | Based on task completion, reasoning trace, tool use  | Based on safety, adaptability, and mission success       |

---

## Related Terms

| Term                  | Description                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| AI Agent              | A program that perceives, reasons, and acts to achieve a goal               |
| Multi-Agent System    | A group of agents interacting to solve distributed or cooperative tasks     |
| Tool-Calling Agent    | An LLM-based agent that invokes external tools or APIs to complete tasks    |
| Memory-Augmented Agent| Agent with persistent memory for context retention and long-term planning   |
| Agentic Workflow      | A structured sequence of steps where an agent reasons, decides, and acts    |
