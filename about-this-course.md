# About This Course  
**Generative AI, Prompt Engineering & Agentic Systems**  
A modular, open-source, instructor-led course designed and delivered by **Satya Prakash Nigam**, Independent AI Consultant.

---

## Course Objectives

1. Introduce learners to the principles and applications of Generative AI, including large language models and multi-modal systems  
2. Equip participants with practical skills in prompt engineering, agentic workflows, and ethical deployment of autonomous AI systems  
3. Foster professional habits in building, evaluating, and sharing GenAI artifacts using reproducible workflows, modular design, and open-source platforms  

---

## Course Overview

- Understand the architecture and capabilities of foundation models (e.g., GPT, Claude, Gemini)  
- Learn prompt engineering techniques for coding, reasoning, search, and creative tasks  
- Explore agentic AI systems: autonomous workflows, tool use, and memory  
- Build mini-projects using open-source GenAI platforms and APIs  
- Discuss emerging trends: retrieval-augmented generation (RAG), multi-modal agents, and AI safety  

---

## Prerequisites

- Basic knowledge of Python programming  
- Familiarity with core AI concepts (e.g., supervised learning, ML pipelines)  
- Personal laptop with stable internet connection  
> Note: Any large files to be downloaded will be communicated in advance.

---

## Learning Outcomes

By the end of this course, learners will:

- Design, evaluate, and deploy GenAI workflows using prompt engineering, tool use, and agentic logic  
- Demonstrate ethical awareness, technical fluency, and professional habits in building autonomous AI systems  
- Create reproducible, documented, and deployable GenAI artifacts for real-world applications

## Hardware & Software Requirements

This course is designed to run entirely on learner-owned laptops with modest configurations. All tools are free, open-source, and compatible with CPU-only systems.

### Minimum Hardware

| Component   | Requirement                                      |
|------------|--------------------------------------------------|
| RAM        | Minimum 4 GB (8 GB recommended)                  |
| Disk Space | Minimum 100 GB free                              |
| Processor  | Any modern CPU (Intel i3/i5/i7, AMD Ryzen, M-series) |
| Internet   | Stable connection (>= 10 Mbps recommended)       |
| Device     | Personal laptop (Windows, macOS, or Linux)       |

---

### Required Software

- Python >= 3.10.x  
- Recommended: Install via [Python.org](https://www.python.org) or Anaconda

#### Python Libraries

```bash
pip install streamlit langchain transformers huggingface_hub pandas matplotlib scikit-learn requests ollama
# Optional (for advanced exploration)
pip install sentence-transformers chromadb openai tiktoken
```

> Note: `openai` is used only for local simulation with dummy keys - no paid API access required.

### Local Model Compatibility

Learners will use lightweight models compatible with CPU-only systems:

| Model Name           | Size     | Use Case                    |
|----------------------|----------|-----------------------------|
| nomic-embed-text     | 274 MB   | Text embedding and similarity |
| tinyllama            | 637 MB   | Prompt experimentation      |
| gemma-2b-sentiment   | 1.7 GB   | Sentiment classification    |
| llama3:8b            | 4.7 GB   | Advanced agentic workflows (optional) |

> These models are pre-loaded via Ollama, a local LLM runtime. Learners will run and interact with these models without cloud dependencies.
