# AI Agents

This repository contains reference implementations of AI Agents built as Streamlit applications.
Faculty-curated agents live here, while students will build their own in `agentic-apps/`.

## Structure
- logs/ : runtime logs
- reports/ : evaluation reports
- datasets/ : sample datasets
- results/ : experiment outputs
- models/ : saved/fine-tuned models
- Each agent folder contains:
  - app.py : Streamlit entry point
  - agent-spec.md : design document
  - workflow.md : execution plan
  - requirements.txt : dependencies
  - prompts/ : reusable prompt files
