# Demonstrational Prompting Template

## Purpose
Provide explicit demonstrations or worked examples for the model to imitate.

## Template Structure
Instruction: [Describe the task]  
Demonstration: [Show example input and output]  
New Input: [Your query]  
Output: [Answer following demonstration style]

## Example Prompt
Instruction: Convert temperatures from Celsius to Fahrenheit.  
Demonstration:  
- Input: 0C → Output: 32F  
- Input: 100C → Output: 212F  
New Input: 25C  
Output: [Expected conversion]
