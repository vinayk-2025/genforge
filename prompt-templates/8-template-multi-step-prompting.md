# Multi-Step Prompting Template

## Purpose
Divide a task into explicit stages with separate prompts.

## Template Structure
Stage 1: [Initial task prompt]  
Stage 2: [Follow-up prompt building on Stage 1]  
Stage 3: [Final prompt combining results]  
Output: [Composed solution]

## Example Prompt
Stage 1: Generate a list of 3 marketing strategies.  
Stage 2: Select the most cost-effective strategy and outline steps.  
Stage 3: Write a 1-paragraph pitch for management.  
Output: Pitch based on selected strategy.
