# Prompt Tuning Template

## Purpose
Adjust prompts systematically to improve model performance across tasks.

## Template Structure
Instruction: [Describe the task]  
Initial Prompt: [Baseline instruction]  
Tuned Prompt: [Refined instruction with constraints or examples]  
Output: [Improved answer]

## Example Prompt
Instruction: Summarise a research paper.  
Initial Prompt: "Summarise the paper."  
Tuned Prompt: "Summarise the paper in 5 bullet points, focusing on methods and results."  
Output: Structured summary.
