# Prompt Chaining Template

## Purpose
Break a complex task into multiple prompts, each building on the previous output.

## Template Structure
Step 1: [Initial task prompt]  
Step 2: [Follow-up prompt using Step 1 output]  
Step 3: [Final prompt combining results]  
Output: [Composed solution]

## Example Prompt
Step 1: Generate a list of 5 startup ideas in education.  
Step 2: Select the most feasible idea and outline a business plan.  
Step 3: Write a 2-paragraph pitch for investors.  
Output: Investor pitch based on selected idea.
