# Zero-Shot Prompting Template

## Purpose
Use when no examples are provided. The model must rely only on the instruction.

## Template Structure
Instruction: [Describe the task clearly]  
Constraints: [Specify language, format, style]  
Output: [Desired form of answer]

## Example Prompt
Instruction: Explain the concept of recursion in simple terms.  
Constraints: Use plain English, no code.  
Output: A short paragraph explanation.
