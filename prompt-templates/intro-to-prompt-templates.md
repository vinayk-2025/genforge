# Prompt Templates Index

This index maps template numbers to their names and provides a short description.  
Use this as a quick lookup when working with domain prompts (e.g., `education-creativity-01-5-prompt.md` → Template 5: Role-Playing).

---

## 1. Zero-Shot Prompting
File: `1-template-zero-shot-prompting.md`  
Description: Direct instruction without examples; useful for simple Q&A or straightforward tasks.

## 2. Chain-of-Thought
File: `2-template-chain-of-thought.md`  
Description: Encourages step-by-step reasoning for complex problems.

## 3. Few-Shot Prompting
File: `3-template-few-shot-prompting.md`  
Description: Provides a handful of examples to guide the model’s output.

## 4. Few-Shot CoT
File: `4-template-few-shot-cot.md`  
Description: Combines few-shot examples with chain-of-thought reasoning.

## 5. Role-Playing
File: `5-template-role-playing.md`  
Description: Model adopts a persona or role to simulate scenarios.

## 6. Tree-of-Thought
File: `6-template-tree-of-thought.md`  
Description: Branching reasoning paths to explore multiple solution routes.

## 7. Prompt Chaining
File: `7-template-prompt-chaining.md`  
Description: Multi-prompt workflows where outputs feed into subsequent prompts.

## 8. Multi-Step Prompting
File: `8-template-multi-step-prompting.md`  
Description: Explicit staged tasks broken into sequential steps.

## 9. Multi-Modal Prompting
File: `9-template-multi-modal-prompting.md`  
Description: Combines text with other inputs (images, audio, etc.).

## 10. Retrieval-Augmented Generation (RAG)
File: `10-template-rag.md`  
Description: Uses external knowledge sources to enrich responses.

## 11. Reflexive Prompting
File: `11-template-reflexive-prompting.md`  
Description: Produces an answer plus a reflection or critique of that answer.

## 12. Self-Consistency
File: `12-template-self-consistency.md`  
Description: Generates multiple reasoning paths and selects the most consistent result.

## 13. Structured Prompting
File: `13-template-structured-prompting.md`  
Description: Enforces schema or format control in outputs.

## 14. Symbolic Prompting
File: `14-template-symbolic-prompting.md`  
Description: Uses metaphors or symbolic framing to explain concepts.

## 15. Thought Source
File: `15-template-thought-source.md`  
Description: Perspective-based reasoning; cites sources of thought.

## 16. Verification Prompting
File: `16-template-verification-prompting.md`  
Description: Fact and constraint checking before finalizing output.

## 17. Analogical Prompting
File: `17-template-analogical-prompting.md`  
Description: Explains concepts by drawing analogies.

## 18. Cognitive Prompting
File: `18-template-cognitive-prompting.md`  
Description: Encourages reflective reasoning and deeper analysis.

## 19. Contrastive Prompting
File: `19-template-contrastive-prompting.md`  
Description: Side-by-side comparison of alternatives.

## 20. Deliberation Prompting
File: `20-template-deliberation-prompting.md`

## 21. Demonstrational Prompting
File: `21-template-demonstrational-prompting.md`  
Description: Provides worked examples to guide responses.

## 22. Error Induction Prompting
File: `22-template-error-induction-prompting.md`

## 23. Exploratory Prompting
File: `23-template-exploratory-prompting.md`

## 24. In-Context Learning

## 26. Meta-Prompting

## Usage Note
When creating prompts in the domain library (`prompt-library/`), suffix filenames with the template number.  
Example: `education-creativity-01-5-prompt.md` → uses Template 5 (Role-Playing).
