# Structured Prompting Template

## Purpose
Force the model to produce outputs in a specific schema or layout.

## Template Structure
Instruction: [Describe the task]  
Format: [Specify table, JSON, list, etc.]  
Output: [Answer in required structure]

## Example Prompt
Instruction: Summarise three programming languages.  
Format: Table with columns: Language, Paradigm, Key Feature.  
Output: Structured table with entries for Python, C, and Java.
