# Multi-Modal Prompting Template

## Purpose
Combine different input types (text, image, table) for richer outputs.

## Template Structure
Instruction: [Describe the task]  
Inputs: [Text + other modality]  
Directive: "Integrate all inputs into the answer."  
Output: [Multi-modal response]

## Example Prompt
Instruction: Summarise the data shown in the table and explain its meaning.  
Inputs: Text description + table of student scores.  
Directive: Integrate both into the answer.  
Output: Summary combining textual explanation and table insights.
