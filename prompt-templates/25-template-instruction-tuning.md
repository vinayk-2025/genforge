# Instruction Tuning Prompting Template

## Purpose
Ensure the model follows explicit directives by reinforcing clarity and format requirements.

## Template Structure
Instruction: [Describe the task clearly]  
Constraints: [Specify format, tone, or layout]  
Directive: "Follow the instructions exactly."  
Output: [Answer aligned with constraints]

## Example Prompt
Instruction: Write a summary of the Industrial Revolution.  
Constraints: Use bullet points, maximum 5 items.  
Directive: Follow the instructions exactly.  
Output: A concise bullet-point summary.
