# Contrastive Prompting Template

## Purpose
Ask the model to compare two or more options side by side.

## Template Structure
Instruction: [Describe the comparison task]  
Directive: "Compare [Option A] vs [Option B]."  
Output: [Side-by-side analysis]

## Example Prompt
Instruction: Compare Python vs Java for beginners.  
Directive: Highlight similarities and differences.  
Output: Table or list comparing ease of use, syntax, applications.
