# Deliberation Prompting Template

## Purpose
Encourage the model to weigh pros and cons before making a decision.

## Template Structure
Instruction: [Describe the decision task]  
Directive: "List pros and cons before choosing."  
Output: [Balanced evaluation + final choice]

## Example Prompt
Instruction: Should students use laptops in class?  
Directive: List pros and cons before choosing.  
Output: Pros (note-taking, resources), Cons (distraction, reduced focus), Final recommendation.
