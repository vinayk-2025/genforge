# Symbolic Prompting Template

## Purpose
Use metaphors or symbolic representations to enrich communication.

## Template Structure
Instruction: [Describe the concept]  
Symbol: "Represent it as [symbol or metaphor]"  
Output: [Answer with symbolic description]

## Example Prompt
Instruction: Explain recursion.  
Symbol: Represent it as a spiral staircase.  
Output: Description using staircase looping infinitely.
