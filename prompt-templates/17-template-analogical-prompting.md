# Analogical Prompting Template

## Purpose
Explain complex ideas by comparing them with familiar concepts.

## Template Structure
Instruction: [Describe the concept]  
Analogy: "Explain using analogy with [familiar domain]"  
Output: [Answer with analogy]

## Example Prompt
Instruction: Explain blockchain.  
Analogy: Compare it to a library record system.  
Output: Explanation using books, records, and catalogues as analogy.
